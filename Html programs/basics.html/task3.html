Dev Environment Overview

LAMP Stack

  • Linux - OS
  • Apache - Web server
  • MySQL - Database
  • PHP - Scripting language
sudo apt update
sudo apt install apache2 mysql-server php libapache2-mod-php php-mysql
sudo systemctl start apache2

WAMP Stack

  • Windows - OS
  • Apache - Web server
  • MySQL - Database
  • PHP - Scripting language
  1. Download WAMP.
  2. Run installer and start WAMP.
  3. Visit http://localhost/.

Apache's Role

Apache handles requests, serves pages, and integrates PHP with MySQL.
